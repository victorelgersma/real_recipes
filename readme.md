# Real Recipes

A no-nonsense recipe site built with Hugo using the PaperMod theme.

## Develop

Build site:

```zsh
hugo
```

add a new post:

hugo new content content/posts/my-first-post.md

index.html file goes under public/

Run development server:

```zsh
hugo serve
```
